## ◈ This Portfolio — Tech Deep Dive

<details>
<summary><b>🏗 Architecture Overview (click to expand)</b></summary>

<br/>

This portfolio site is built as a **React + TypeScript SPA** using Vite, with a heavy emphasis on cinematic UI and 3D interactions.

```
Portfolio/
├── src/
│   ├── pages/          # Full-page sections (Home, About, Projects, Skills, Education, Contact)
│   ├── components/     # Reusable UI atoms
│   │   ├── GlobalMeshBackground.tsx   # Three.js mesh background
│   │   ├── SkillsUniverse.tsx         # Interactive 3D orbital skill system
│   │   ├── CinematicCursor.tsx        # Custom cursor with trail
│   │   ├── Preloader.tsx              # Entry animation
│   │   ├── Navbar.tsx
│   │   └── Magnetic.tsx               # Magnetic hover effect
│   ├── lib/            # Utilities & helpers
│   └── main.tsx
├── public/             # Static assets (images, icons)
├── vite.config.ts
└── vercel.json
```

**Key design decisions:**
- `@react-three/fiber` + `@react-three/drei` for WebGL 3D scenes
- `framer-motion` for layout animations & page transitions
- `gsap` for timeline-based cinematic sequences
- `tailwindcss v4` with custom utility classes
- `@emailjs/browser` for the contact form (no backend needed)
- `@google/genai` for embedded AI features

</details>

## ◈ Run Locally

**Prerequisites:** Node.js ≥ 18

```bash
# 1. Clone the repo
git clone https://github.com/AlinsBinuP/Portfolio.git
cd Portfolio

# 2. Install dependencies
npm install

# 3. Configure environment
cp .env.example .env.local
# → Add your GEMINI_API_KEY to .env.local

# 4. Start the dev server
npm run dev
# → http://localhost:3000
```

| Command | Action |
|---------|--------|
| `npm run dev` | Start dev server at `localhost:3000` |
| `npm run build` | Build for production |
| `npm run preview` | Preview production build |
| `npm run lint` | Type-check with TypeScript |
